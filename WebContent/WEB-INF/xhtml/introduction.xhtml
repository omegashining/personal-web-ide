WEB-IDE es un sistema capaz de brindar a los usuarios programadores una herramienta en línea que permite el desarrollo de aplicaciones en lenguaje C o JAVA, específicamente aplicaciones que no conlleven el uso de componentes gráficos, es decir, que las entradas y salidas sean capturadas y mostradas mediante la terminal del sistema.


Más específicamente el sistema permite:
  • La creación de código fuente en lenguaje C o JAVA, así como también la carga de archivos que el usuario tenga previamente hechos, no necesariamente en el sistema para efectos de edición y/o compilación.
  • Efectuar tareas de compilación mediante una petición de un recurso remoto que provendrá de algún usuario conectado al sistema, en caso de que no sea necesaria la búsqueda de dicho recurso, entonces la compilación se llevara a cabo en el mismo servidor.
  • Mostrar el resultado de la ejecución de un programa previamente compilado resultante dentro del mismo entorno, o si es el caso, despliega los errores sintácticos que se presenten durante la compilación del programa.
  • Dar la opción al usuario de guardar el código fuente que haya generado independientemente de que se haya compilado o no.
  • La descarga del código fuente y el archivo ejecutable generados en el sistema.
  • Evaluar la carga de trabajo que exista en un momento determinado para efectuar la búsqueda de un recurso remoto en la tarea de compilación.
